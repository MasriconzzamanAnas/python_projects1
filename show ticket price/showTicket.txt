# প্রশ্ন
# সিনেমার টিকেটের দাম নির্ধারণ 

# বিষয়বস্তু:আপনাকে একটি প্রোগ্রাম তৈরি করতে হবে যা সিনেমা টিকেটের দাম নির্ধারণ করবে। প্রোগ্রামটি ব্যবহারকারীর বয়স এবং শোয়ের সময় অনুযায়ী নির্দিষ্ট নিয়ম অনুসরণ করে টিকেটের দাম হিসাব করবে। এছাড়া, বিকাল ৫টার আগের শো'র জন্য বিশেষ ছাড়ও প্রদান করতে হবে।


 # টাস্ক: আপনার টাস্ক হলো এমন একটি পাইথন প্রোগ্রাম তৈরি করা যা ব্যবহারকারীর থেকে বয়স এবং শোয়ের সময় ইনপুট নিয়ে টিকেটের দাম নির্ধারণ করবে। প্রোগ্রামটি দনিচের নিয়মাবলী অনুযায়ী কাজ করবে:


 # টিকেটের দাম নির্ধারণের শর্তাবলী:
# 1.   বয়স অনুযায়ী টিকেটের দাম:
    # o    ১০ বছরের সমান বা নিচে: ৩০০ টাকা
    # o    ১১-২৫ বছর: ৫০০ টাকা
    # o    ২৬-৬০ বছর: ৮০০ টাকা
    # o    ৬০ বছরের উপরে: ৪০০ টাকা

# 2.   শোয়ের সময় অনুযায়ী ছাড়:
    # o    যদি শোয়ের সময় বিকাল ৫টার (১৭০০) আগে হয়, তবে টিকেটের মূল দামের উপর ১০% ছাড় প্রযোজ্য হবে।


# 3.   বৈধ ইনপুট নিশ্চিত করা:
    # o    বয়স অবশ্যই ধনাত্মক পূর্ণসংখ্যা হতে হবে।
    # o    শোয়ের সময় অবশ্যই ২৪-ঘন্টার ফরম্যাটে (HHMM) সঠিকভাবে ইনপুট দিতে হবে।


# ইনপুট ফরম্যাট:

# 1.       বয়স: একটি ধনাত্মক পূর্ণসংখ্যা, যেমন ২৫, ৪০ ইত্যাদি।
# 2.       শোয়ের সময়: ২৪-ঘন্টার ফরম্যাটে (HHMM) সময়,প্রথম ২ ডিজিট ঘন্টা পরের  ২ ডিজিট মিনিট প্রকাশ করবে। ঘন্টা মিনিটের মাঝে কোন স্পেস বা ক্যারেক্টার নাই।    যেমন সকাল ৭ টা ১০ মিনিট -> ০৭১০,  বিকাল ৩টা ৩০ মিনিট -> ১৫৩০, সন্ধ্যা ৬টা ৪৫ মিনিট -> ১৮৪৫ ইত্যাদি।


 # আউটপুট ফরম্যাট:
# প্রোগ্রামটি টিকেটের দাম প্রিন্ট করবে। যদি ছাড় প্রযোজ্য হয়, তবে ছাড়কৃত দামসহ আউটপুট দেখাবে।
# Sample Inputs and Outputs:
# Sample 1:
# Input: 
# Age: 8
# Showtime (HHMM): 1630
# Output: 
# Ticket price: 300 BDT
# Discount: 30.00 BDT
# Discounted price: 270.00 BDT
